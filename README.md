# Real-Time Russian Sign Language Recognition
Computer vision model for russian sign language recognition.
## Summary
<br>**Dataset:** https://www.kaggle.com/datasets/kamillakabardieva/russian-sign-language-alphabet
<br>**Model:** Basic CNN (70k parameters)
![alt text](images/model.png)
<br>**Results:** 99.8% AUROC, 90% accuracy on test data
![alt text](images/auroc.png)
<br>**Stack**: Python, PyTorch (Torchvision, Torchmetrics), OpenCV, MLFlow
